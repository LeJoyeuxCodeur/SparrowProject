# === PROJET SPARROW ===


# === But du jeu ===
Jeu solo. 
Le héros commence le jeu avec un bateau simpliste et doit l'améliorer au
fil du jeu. Pour celà, il devra parcourir les océans à la recherche de butin et de nouveaux
horizons. Le jeu se sauvegardera manuellement.


# === LE BATEAU ===
Un bateau possède une barre de durabilité (qui diminue suivant les attaques et les mouvements en mer)

Il pourra posséder une barre de direction, des canons, un équipage, un nombre de mats, et de voiles.
Il sera limité en slots pour ces objets (ex: pas plus de 5 mats)
Il comprendra une soute limitée en capacité.


# === LA CARTE ===
La carte du jeu comprendra plusieurs zones, et ces zones auront leur propre type.

- Mer : Notion de sens du vent et de km/h --> change ces données aléatoirement toutes les x minutes
- Terre : Amarrer le bateau, ravitailler quand cela est possible
